Life of the Party

To connect to the light (MAC):
    select the moodL from the wireless networks
    open networks
    set IPv4 addr: 192.168.0.2
    set subnet mask: 255.255.255.0
    set router: 192.168.0.2
    configure IPv6: link-local only

Test the connection:
    ping 192.168.0.3

Test lights:
    python (lightscript)
